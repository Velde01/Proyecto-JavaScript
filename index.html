Simulador de búsqueda y compra de auto

Este simulador permite al usuario elegir un vehículo disponible para alquilar y calcular el costo total según la cantidad de días seleccionados. El programa utiliza JavaScript para gestionar la interacción con el usuario mediante prompt, confirm y alert, además de aplicar funciones, ciclos y condicionales para procesar los datos ingresados
